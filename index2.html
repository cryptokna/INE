DETAILS

How did the concept of OSTRAN came about?

The idea of OSTRAN came about after we conducted a survey regarding the difficulties that people face when traveling.

To view the results/responses of the survey, click here


What is ili?

Ili is an offline wearable translation device which can translate any input language quickly in a matter of seconds, before outputting it through text to speech(TTS).

Why use ili with the OSTRAN engine?

Since the original STREAM engine for ili is not open source and the developers have no plans to add in more language support, with the help of the OSTRAN engine, ili will be able to have more language support by allowing external developers or the ili community to push updates to github. Most importantly, OSTRAN engine is FREE.


INSTALL

How to install the engine?

To install the engine, you have to download and install the modified version of the official ili desktop application from our github page. After doing so, launch the application and all you have to do is just select "OSTRAN" under "Engine selection".


How to update the engine?

When there is a new commit in our github's master branch, the application will automatically prompt you to update. To update the engine, all you have to do is simply click on the "Update" button and if there is an udpdate, a popout will appear to prompt if you want to update the engine. Click "yes" and wait till it finishes.


Wouldn't this mean that I'll have to update regularly?

No. We will only ENSURE that major updates will be committed in the github page.

F.A.Q.S

How do I contribute if I am a developer?

You can contribute to the engine by heading over to our github page.


How do I get my work to be approved?

To get your work to be approved, we have guidelines which you will be able to find on the github page.



Can I contribute by improving how the engine work?

Yes, you can. Just submit your work on our github page and we will review it. If it is something that can help to improve the engine further, we will approve your commit to the master branch.



Do I get anything if I contribute?

No. By doing this, you are agreeing to give your own time and effort in helping the community to add in more language support. Please do not contribute in hopes of getting anything in return.
